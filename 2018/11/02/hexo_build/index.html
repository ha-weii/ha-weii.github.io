Hexo使用笔记

注意：

1,本文假设你已经安装了Node.js和Git

2,本文的演示适应于windos系统，其他系统也可以参考

3,安装Node.js和Git请移步

1.安装hexo

打开git bash,使用下面命令安装hexo

npm install -g hexo-cli        # -g表示全局安装

2.初始化

建立hexo文件夹用于存放博客(F:\hexo),进入hexo文件夹，使用右键打开git bash

hexo init [folder]         # 如果没有跟[folder],那么默认在当前文件夹建立网站

此时在hexo文件夹下已经有一些文件

关于这些文件的说明请移步

3.安装启动hexo-server

npm install hexo-server --save
hexo server

此时可以访问服务器，默认情况下访问http://localhost:4000/,如果访问不了尝试改变端口,使用-p 重设端口 hexo server -p 5000

4.生成文件并部署

hexo generate

在部署之前必须制定部署地点必须先在 _config.yml 中修改参数，一个正确的部署配置中至少要有 type 参数，例如：

deploy:
   type: git

   repository: https://github.com/xxx/xxx.github.io.git

   branch: master

安装hexo-deployer-git


npm install hexo-deployer-git --save

部署

hexo generate --deploy #简写为hexo g -d
